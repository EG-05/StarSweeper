# StarSweeper

A combination of mine-sweeper logic where each star is stable/unstable/dead 

Clicking on a stable star allows you to connect the line to another dot to form an constellation 
Clicking on an unstable star takes you into a fast-paced mini-game where you're an astronaut avoiding asteroids, survive to make the star stable! 
Clicking on a dead star/unable to finish the mini game - Game over 

A game designed whose core is based on Object Oriented Programming, using the Allegro Library 
Implementing OOP concepts: Abstraction, Encapsulation, Inheritance, and Polymorphism

Project completion date: April 2025
